### 应用介绍
本应用用以查询疫情相关信息（各国疫情数据统计，各省疫情数据统计，最新新闻等）；客户端从服务器拉取新冠相关数据并实现相关功能，包括新冠新闻获取、展示和管理，全球疫情数据的展示、更新等。
